# Adapters Module

This module provides the abstraction layer for Model Providers (LLMs).

## Responsibilities
- Implement the `BaseModelAdapter` interface.
- Handle API authentication and connection details.
- Standardize model inputs and outputs.
- Manage rate limits and retries.

## Supported Providers
- (Future) OpenAI
- (Future) Anthropic
- (Future) HuggingFace / Local
