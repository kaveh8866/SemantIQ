# SMF Question Sets Changelog

## v1.0 (Initial Release)
- **Release Date:** 2025-12-28
- **Scope:** Initial implementation of 12 core categories and 3 extensions.
- **Coverage:** 5 questions per category minimum.
- **Methodology:** Based on SMF v1.1 Registry and Question Archetypes v1.0.

### Added Categories
- **Core:**
    - `mcb` (Meaning Coherence)
    - `cib` (Context Integration)
    - `ncb` (Nuance & Complexity)
    - `rfb` (Reasoning & Formal Logic) - *Pending generation*
    - `ras` (Robustness) - *Pending generation*
    - `elb` (Ethical) - *Pending generation*
    - `ssb` (Social) - *Pending generation*
    - `mlrb` (Multilingual) - *Pending generation*
    - `ccb` (Creativity) - *Pending generation*
    - `lfcb` (Long-Form) - *Pending generation*
    - `ssb2` (Symbolic) - *Pending generation*
    - `qgb` (Grounding) - *Pending generation*
- **Extensions:**
    - `dtb` (Domain Technical)
    - `msb` (Multimodal)
    - `awm` (Agentic Workflow)
