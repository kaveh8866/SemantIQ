# Citation & DOI Guide

## How to Cite SemantIQ

If you use SemantIQ-M-Benchmarks in your research, please use the following citation. This helps us track impact and ensures reproducibility.

### General Software Citation
Use the following BibTeX for the framework itself:

```bibtex
@software{semantiq_benchmarks,
  author = {{SemantIQ Research Team}},
  title = {SemantIQ-M-Benchmarks: A Unified Framework for Multimodal AI Evaluation},
  year = {2025},
  url = {https://github.com/kaveh8866/SemantIQ},
  version = {0.1.0}
}
```

### Dataset Citations
When referring to specific benchmark datasets, please include the version used.

**SMF (Semantic Maturity Framework)**
```bibtex
@dataset{semantiq_smf,
  author = {{SemantIQ Research Team}},
  title = {Semantic Maturity Framework (SMF) Dataset},
  version = {1.0},
  year = {2025},
  url = {https://github.com/kaveh8866/SemantIQ/tree/main/datasets/smf}
}
```

**HACS (Human-AI Cognitive Symmetry)**
```bibtex
@dataset{semantiq_hacs,
  author = {{SemantIQ Research Team}},
  title = {HACS: Human-AI Cognitive Symmetry Benchmark},
  version = {1.0},
  year = {2025},
  url = {https://github.com/kaveh8866/SemantIQ/tree/main/datasets/hacs}
}
```
