# Reporting Standards

All research reports generated by SemantIQ-M must include:

## 1. Metadata
- **Study ID**: Unique identifier (UUID).
- **Date**: ISO 8601 format.
- **Git Hash**: Commit hash of the codebase.
- **Benchmark Version**: Version of the dataset used (e.g., SMF v1.0).

## 2. Methodology
- **Raters**: Number of raters, recruitment method (if public).
- **Models**: List of models evaluated.
- **Sampling**: Number of prompts, selection method.

## 3. Results
- **Descriptive Statistics**: Mean, SD, Median for each model/criterion.
- **Reliability**: ICC or Alpha values for all human ratings.
- **Significance**: P-values for pairwise comparisons (if applicable), with correction for multiple comparisons (e.g., Bonferroni).

## 4. Visualizations
- **Radar Charts**: For multidimensional comparison.
- **Box Plots**: To show score distributions.
- **Correlation Matrices**: For validity checks.

## 5. Appendices
- **Data Manifest**: Link to raw data files.
- **Prompt List**: Full text of prompts used (or link to file).
